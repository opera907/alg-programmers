## 기능 작동 완성

[내 로직]
1. new Date() 객체 만듦 
2. 원하는 날짜 생성 => 2016년과, 매개변수로 받은 월과 일을 new Date() 의 인수로 전달하여 새 인스턴스에 넣음
3. 원하는 날짜의 요일 인덱스 받음
4. days = ['SUN','MON','TUE','WED','THU','FRI','SAT']  
3의 인덱스로 해당 요일의 문자열 출력
